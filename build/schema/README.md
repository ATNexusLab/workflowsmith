# Schema

This directory contains canonical schema definitions for WorkflowSmith workflow units.

## What Is Here

- `workflow-unit.template.md` — the canonical template for any workflow unit. Copy this file when creating a new agent, skill, policy, checklist, or memory index.

## What Is Not Here Yet

Additional schema definitions (field validation rules, type-specific required fields, adapter mapping contracts) will be added in later phases as the content layer matures.

## Schema Authority

The field specification is defined in [ADR-001](../../docs/decisions/ADR-001-canonical-schema.md). If the template and the ADR conflict, the ADR is authoritative.
