# Workflow Spec

This directory contains canonical framework specifications for WorkflowSmith.

Spec documents define:

- the resource model harnesses must satisfy
- the types of canonical workflow units
- authoring rules and size budgets for workflow content

These documents are pre-requisites for authoring the canonical workflow. They
are not workflow content themselves.

## Documents

- [Canonical Harness Resource Model](harness-resources.md) — complete resource
  contract for modern AI software engineering harnesses.
- [Instruction, Agent, And Skill Model](instruction-agent-skill-model.md) —
  canonical boundaries for instructions, agents, and skills.
- [Authoring Modularity](authoring-modularity.md) — size budgets and split rules
  for canonical workflow units.
- [Automation Policy](automation-policy.md) — scoped use of formatters, linters,
  autofix, and equivalent repair tools during workflow authoring.
