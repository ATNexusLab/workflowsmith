# Workflow Source

This directory contains the canonical WorkflowSmith source.

The 0.0.0 milestone defines structure only. Complete workflow content is authored in later milestones through the RFC/ADR development process.

## Directories

- `source/` - canonical workflow units.
- `spec/` - canonical framework specifications (harness resource model, unit types, authoring rules).
- `schema/` - schema and authoring expectations.
